Smart Yatra - Public Transport Digitization System

Features:

- Real-Time Bus Tracking
- Route Optimization
- QR Code Payments
- Live Notifications
- Admin Dashboard

Passenger Panel Pages

- Live Bus Tracking Page:

  - Map view showing buses in real-time.
  - Estimated time of arrival (ETA) for selected routes.
  - Bus details (e.g., capacity, fare, route number).

- Route Planner Page:

  - Input starting point and destination.
  - Suggest optimal routes with estimated time and fare.
  - Alternative route suggestions based on traffic.

- QR Code Payment Page:

  - Display fare details.
  - Option to scan a QR code for payment.
  - Payment confirmation and receipt generation.

- My Rides Page:

  - List of recent rides taken.
  - Fare details and ride history.

- Notifications Page:

  - Real-time notifications for bus delays, cancellations, or route changes.

- Passenger Dashboard Page:
  - Profile settings (edit user info).
  - Wallet/fare balance.
  - Review your feedback on rides.

Driver Panel Pages

- Driver Dashboard Page:

  - Overview of active routes, estimated traffic.
  - Today's earnings and completed trips.

- Real-Time Bus Location Page:

  - Map showing the live route and traffic updates.
  - Route optimization suggestions based on traffic conditions.

- Fare Management Page:

  - Collect fare via QR codes.
  - View fare summary.

- Notification Page:

  - Updates about route changes or traffic alerts.

- Profile Page:
  - Edit profile, view personal stats, etc.

Admin Panel Pages

- Admin Dashboard Page:

  - High-level overview of all buses, routes, and operations.
  - Key metrics: total rides, earnings, and passenger feedback.

- Route Management Page:

  - Create, edit, or delete bus routes.
  - Assign buses and drivers to routes.
  - View real-time traffic on routes.

- Fare Management Page:

  - Set fare prices.
  - View fare collection statistics and financial reports.

- Bus Management Page:

  - Add, update, or remove buses from the fleet.
  - Assign buses to specific routes.

- Driver Management Page:

  - Add, edit, or remove drivers.
  - View driver performance and stats.

- Passenger Management Page:

  - Manage passenger accounts.
  - View passenger ride history and complaints.

- Notification Management Page:

  - Set up dynamic notifications for passengers and drivers.

- Analytics Page:
  - View trends in bus usage, busiest routes, passenger satisfaction.
  - Download reports for operational improvements.

Done:

- 404/NotFound Page: Displayed when users navigate to an invalid page.
- Login / Sign-up Pages: Separate login/sign-up flows for drivers, passengers, and admins.
- Help/Support Page: About, contact information, etc.
- Settings Page: Update personal information, notification preferences, etc.

Passenger Panel Pages

- Home Page:

  - Display real-time bus locations on a map.
  - Search bar for finding routes and buses.
  - CTA for login/sign-up.
